# news

#### 學習目標:
  - 透過實作一個新聞網站，來練習Servlet與AJAX技術。

#### 使用技術:

  - 資料庫: [SQL Server](https://www.microsoft.com/en-us/sql-server/)
  - 第三方API: [Bootstrap5](https://getbootstrap.com/)
